# Code of Conduct

This repository is intended to be practical, respectful, and high-signal.

## Expected Behavior

- be accurate
- be direct
- cite sources
- correct errors without drama
- keep discussions focused on maintainability and engineering usefulness

## Unacceptable Behavior

- knowingly adding hallucinated commands
- misrepresenting vendor capabilities
- personal attacks
- low-effort marketing language instead of technical clarity

## Enforcement

Maintainers may edit, close, or reject contributions that reduce accuracy or increase confusion.
