# Quick Start

## Fast Path

1. Read [Choosing the Right Tool](choosing-the-right-tool.md).
2. Pick one terminal agent and one IDE agent.
3. Copy [../templates/AGENTS.md](../templates/AGENTS.md) into your repo.
4. Review [../governance/security-and-permissions.md](../governance/security-and-permissions.md).
5. Start with planning and review workflows before broad automation.

## Team Default Recommendation

- terminal-first: Claude Code or OpenAI Codex
- IDE-first: Cursor or GitHub Copilot
- open-source path: Aider, Continue, or OpenCode
- provider diversity: Gemini CLI or xAI integrations
