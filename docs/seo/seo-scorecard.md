# SEO Scorecard

Track these metrics monthly after the first 28-day baseline is available.

## Search Metrics

- indexed pages
- search impressions
- search clicks
- CTR
- average position
- top queries
- top landing pages
- branded vs non-branded traffic

## Developer Engagement

- GitHub repository clicks
- stars
- forks
- clones
- contributors
- visits to tool pages
- visits to prompt and template pages

## Organization Engagement

- visits to governance pages
- visits to audience landing pages
- visits to MCP approval policy
- visits to team rollout guide
- visits to tool evaluation template
- internal-fork-oriented traffic signals

## Content Quality

- pages lacking unique descriptions
- pages lacking sources where required
- pages marked `Needs verification`
- broken links
- stale verification dates
- orphan pages
- low-engagement pages with strategic importance

## Suggested Review Cadence

- monthly: search and engagement
- quarterly: metadata refresh, stale verification review, topic-map update
