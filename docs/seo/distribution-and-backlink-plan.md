# Distribution and Backlink Plan

Use distribution to extend reach around genuinely useful technical content, not to spam communities.

## Developer Channels

- GitHub Awesome lists relevant to AI developer tools and documentation
- Dev.to or Hashnode posts summarizing:
  - AI coding tool comparison
  - AGENTS.md guidance
  - MCP security basics
  - AI code review workflows
- relevant Reddit, Discord, and Slack communities when the post adds standalone value
- LinkedIn and X posts aimed at developers and engineering leads

## Organization Channels

- developer-experience communities
- platform engineering communities
- security and AppSec communities
- engineering leadership newsletters
- company engineering blogs
- conference talks and meetup talks about AI coding governance and internal handbook design

## Reusable Campaign Themes

- AI coding tools comparison for teams
- AGENTS.md and repository instruction files
- MCP security and approval
- AI code review and bug-fixing workflows
- internal AI engineering handbook templates

## Outreach Rules

- tailor each post to the community
- link to the strongest relevant page, not always the homepage
- provide practical context in the post itself
- never buy low-quality backlinks
