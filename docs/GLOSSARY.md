# Glossary

## Agentic Coding

Coding workflows where the tool can reason, plan, edit, and sometimes run commands or external tools.

## CLI Agent

A coding agent that primarily runs in the terminal.

## IDE Agent

A coding assistant embedded in an editor or IDE.

## MCP

Model Context Protocol, a standard for connecting models to tools and context sources.

## Instruction File

A persistent guidance file such as `AGENTS.md`, `CLAUDE.md`, `copilot-instructions.md`, or `.cursor/rules`.
